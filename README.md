# Analog-watch
This watch shows current time ,changes per sec  
The Analog Watch project is a web-based clock built using HTML, CSS, and JavaScript. It displays a real-time analog watch with moving hour, minute, and second hands, simulating a traditional clock face.

# Features
Real-time clock functionality

Smooth movement of hour, minute, and second hands
Responsive design for different screen sizes
Lightweight and easy to run in any browser

#Technologies Used

HTML5 – for structure
CSS3 – for styling the clock face and hands
JavaScript (ES6) – for dynamic time updates
